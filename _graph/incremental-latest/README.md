# Published Graph Snapshot

This folder is a controlled, commit-safe Graphify snapshot generated from the
RCM AI Knowledge Base.

Read order for AI agents:

1. `AGENTS.md`
2. `READ_PROTOCOL.md`
3. `AI_PROTOCOL.md`
4. `index.json`
5. `_graph/GRAPH_REPORT.md`
6. `_graph/incremental-latest/GRAPH_REPORT.md` when the task depends on recent changes
7. `_graph/graph.json` and `_graph/incremental-latest/graph.json` only for navigation
8. Source `meta.json`, `report.md`, `lessons.md`, `challenges/`, and `patches/`

Graph scopes:

- `_graph/` is the full advisory snapshot when intentionally refreshed.
- `_graph/incremental-latest/` is the recent-change snapshot generated by the active production runner.
- The active runner, schedule, model, and notification contract are documented in `docs/graphify-production-operations.md`.
- The exact backend and model for this snapshot are recorded in `manifest.json`.

Rules:

- `_graph/` is advisory only.
- `_graph/incremental-latest/` is not a replacement for the full graph or `index.json`.
- Do not treat inferred graph edges as KB truth.
- Do not edit active KB entries based only on graph output.
- Convert substantive graph findings into a challenge, patch, or governed KB entry.
- Raw `graphify-kb-corpus/`, `graphify-kb-corpus-incremental/`, `.graphify-kb-corpus/`, and `graphify-out/` stay local and ignored.
